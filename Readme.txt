How to run:
  1. open the executable files.
  2. start the server first by clicking the start button.
  3. put username in the client UI and click connect.
  4. open another client and repeat step 3.
  5. now you can send and recieve message.
  6. I give 5 space in the thread. 
  7. if sixth client asks for access he will be sent to waiting room, if any of the previous two clients disconnects then sixth           client can join the server.
  8. enjoy chatting.
